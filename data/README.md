# Data

This folder contains compact data needed to understand or regenerate the main
paper outputs.

- `derived_maps/`: final vigour-network weight map, p90 HTML overlay, and
  standard-GLM task-activation z map.
- `model_ablation/`: ablation maps and SLURM logs used for Figures 4 and 5.
- `metadata/`: study metadata files copied from the working repository.
- `processed/gvs_connectivity/`: processed ROI beta and run-inventory tables
  used by the GVS connectogram analyses.
- `external/`: notes for subject-level inputs that are not packaged here.

The full subject-level beta and behaviour dataset is not included in this
curated repo.
