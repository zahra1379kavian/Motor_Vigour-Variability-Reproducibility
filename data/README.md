# Data

This folder contains compact data included with the curated reproducibility
package. It is not a full subject-level study dataset.

- `derived_maps/`: final vigour-network weight map, p90 HTML overlay, and
  standard-GLM/task/GLMsingle maps used by the main and supplementary figures.
- `metadata/`: study metadata files included for provenance.
- `processed/gvs_connectivity/`: processed ROI beta and run-inventory tables
  used by the GVS connectogram analyses.
- `external/`: notes for subject-level inputs that are not packaged here.

The full subject-level beta, behavioural, BOLD, and large intermediate
ablation datasets are not included in this repository.
