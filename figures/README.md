# Figures

`main/` contains the current paper main figures using publication-facing file
names. `supplementary/` contains the supplementary figure exports.

Each figure is stored as both PNG and PDF when the source export had both
formats.

`main/Figure1.pdf` is the graphical overview of the behavioural task and the
constrained optimization framework. `main/Figure1.png` is a raster export of the
same figure (200 DPI) used as the overview image on the repository front page.
